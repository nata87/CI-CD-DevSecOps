# 🛡️ CI/CD & DevSecOps: Лабораторна робота №8

Цей репозиторій містить демонстраційний застосунок на Python та конфігурацію GitLab CI/CD для демонстрації інтеграції безпекових перевірок у процес розробки.

**Основні реалізовані концепції:**

* **Повний DevSecOps пайплайн:** Налаштовано автоматизовані перевірки на кожному етапі життєвого циклу (Secret Scan, SAST, Unit Tests, SCA, DAST).
* **Принцип Fail Fast:** Пайплайн налаштовано на примусову зупинку (`allow_failure: false`) при провалі функціональних тестів або виявленні вразливостей. Це запобігає розгортанню небезпечного коду та економить ресурси CI-ранерів.
* **Quality Gates:** Успішне проходження модульних тестів є обов'язковим першим шлюзом перед початком збірки Docker-образу та динамічним аналізом (DAST).
* **Аналіз залежностей:** Реалізовано сканування контейнерів та залежностей на наявність відомих CVE за допомогою інтегрованих сканерів GitLab та Trivy.
